# Welcome to your CDK TypeScript project for Lambda Insights.


## Useful commands

 * `cdk bootstrap`   CDK Bootstrapping
 * `cdk synth`       Emits the synthesized CloudFormation template
 * `cdk deploy`      Deploy the Stack
